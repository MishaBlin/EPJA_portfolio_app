# EPJA 2024 (Team Cats)

## Launch instructions

### 1. Clone repository

```shell
git clone https://github.com/MishaBlin/EPJA_portfolio_app.git
cd EPJA_portfolio_app
```

### 2. Install dependencies

```shell
npm ci
```

### 3. Create `.env` with content of `.env.example`:

```
TOKEN=ASDFGHJKLLKJHGFDSDFGHJKJHGF
```

### 4. Launch dev build

```shell
npm run start
```
